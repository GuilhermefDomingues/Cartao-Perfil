# Profile Card UI

Projeto desenvolvido com HTML e CSS que cria um cartão de perfil moderno e responsivo. A interface exibe informações pessoais, redes sociais, botões de interação e estatísticas de engajamento em um layout elegante.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Google Fonts (Poppins)

## ✨ Funcionalidades

- Foto de perfil
- Nome do usuário
- Cargo ou profissão
- Links para redes sociais
- Botões de ação
- Estatísticas de engajamento
- Efeito hover
- Layout responsivo

## 📂 Estrutura do Projeto

```bash
Profile-Card-UI/
│
├── index.html
├── css/
│   └── style.css
├── img/
│   └── profile.jpg
└── README.md
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.
